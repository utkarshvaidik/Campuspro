# Campuspro Backend (हिंदी गाइड)

## कैसे चलाएं?
1. टर्मिनल में जाएं, backend फोल्डर में जाएं  
2. `npm install` चलाएं (dependencies इंस्टॉल होंगी)
3. `npm start` चलाएं  
4. API http://localhost:4000 पर चलेगी

## यूज़र डेमो लॉगिन
